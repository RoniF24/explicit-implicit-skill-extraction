# SkillSight

Shared repo for the “Explicit-Implicit Skill Extraction” project.

## Structure
- src/        core code
- scripts/    runnable scripts
- notebooks/  experiments / EDA
- docs/       slides & reports
- data/       local data (not committed)

## Setup
python -m venv .venv
# Windows:
.venv\\Scripts\\activate
pip install -r requirements.txt
